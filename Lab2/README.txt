This project creates a fork that creates a child process. Both the child and parent 
process read each line from the file and display it to the screen along with their 
process ID number.

Compile and execution instructions:
Make sure that the path of the file you want to read is correct. It will varry from
machine to machine. Other than that there is no command-line arguments, or special 
compile instructions.
